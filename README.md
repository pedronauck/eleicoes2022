## Conclusão

Matematicamente, seria praticamente nulo a possibilidade de variação tão grande usando como base apenas o modelo de urna. Um modelo de urna não deveria influenciar o voto de uma pessoa. Quando visto que essa diferença ocorre apenas nas regiões Norte e Nordeste e que ela aumenta conforme vai se diminuindo o número de habitantes de uma região, só fortalece a tese de que houve possível manipulação algorítmica nos resultados.
